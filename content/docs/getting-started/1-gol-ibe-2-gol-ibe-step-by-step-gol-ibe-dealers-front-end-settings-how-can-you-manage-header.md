# How can you manage the header on your GOL IBE web?

<!-- tags: header, navigation, menu, front-end, branding, dealer -->

The **header** is the top bar of your GOL IBE booking engine front-end. It typically contains your logo and navigation links. You can customise its content using Supporting Texts.

## What you can change in the header

| Element | How to change |
|---------|--------------|
| Logo | **Dealers → Front-end settings → Banner** |
| Logo link (clicks to your website) | **Dealers → Front-end settings → URL Homepage** |
| Navigation links / menu | **Supporting texts → Supporting texts** → menu type |

## How to add navigation links to the header

1. Log into the GOL IBE admin console.
2. Go to **Supporting texts → Supporting texts**.
3. Click **Add text**.
4. Set the **Text type** to the appropriate menu type.
5. Add your links in the **Body** field.
6. Click **Save** and go to **Code Lists → Flush HTML Caches**.

## How to change header colours

Use the **Background** section in **Dealers → Front-end settings** to adjust colours by clicking on the coloured boxes. For more advanced styling, go to **Supporting texts → CSS** and add your custom CSS rules.
