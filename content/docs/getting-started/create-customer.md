# Create a Customer Profile

<!-- tags: customer, create, passenger, profile, new -->

Saving a customer profile speeds up future bookings — passenger details are stored in the system for reuse.

## Steps

1. Go to **Customers → Customers**.
2. Click **Add customer** (or **New customer**).
3. Enter passenger details:
   - Title, first name, last name
   - Date of birth
   - Contact email and phone
4. Optionally add **travel documents** (passport, ID card).
5. Click **Save**.
