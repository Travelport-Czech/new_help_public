# Users

<!-- tags: users, agents, rights, permissions, passwords, access -->

The **Users** section lets you manage all staff accounts that have access to the GOL IBE admin console. Each user gets their own login credentials and access rights that define what they can see and do.

## What you can do here

- **Create** new agent accounts with specific access rights
- **Assign rights** — control what sections and actions each user can access
- **Reset passwords** for agents who have been locked out
- **Manage your own account** — update your profile and password

Keeping user accounts up to date ensures that only authorised staff can access your GOL IBE admin console.

> **Security note:** For security reasons, passwords should be changed at least every 3 months.
