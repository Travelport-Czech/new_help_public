# How to set up a working hours exception

<!-- tags: working hours, exception, holiday, public holiday, closure, schedule -->

Working hours exceptions let you override your regular schedule for specific dates — for example, to close on a public holiday or set different hours for a particular day.

## How to add an exception

1. Log into the GOL IBE admin console.
2. Go to **Agency → Working hours modifications**.
3. Add the date you need to define as an exception.
4. Choose:
   - **Date only (no time)** — the agency is closed for the whole day
   - **Date with Since and Till times** — the working hours on that day are limited to the defined period
5. Save changes.

> There is no built-in code list for public holidays. If you need to define these days, please do so manually.
